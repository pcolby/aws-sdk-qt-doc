103 /**
104  * @class CodeCommitClient
105  *
106  * @brief Client for AWS CodeCommit
107  *
108  * <fullname>AWS CodeCommit</fullname>
109  *
110  * This is the <i>AWS CodeCommit API Reference</i>. This reference provides descriptions of the operations and data types
111  * for AWS CodeCommit API along with usage
112  *
113  * examples>
114  *
115  * You can use the AWS CodeCommit API to work with the following
116  *
117  * objects>
118  *
119  * Repositories, by calling the
120  *
121  * following> <ul> <li>
122  *
123  * <a>BatchGetRepositories</a>, which returns information about one or more repositories associated with your AWS
124  *
125  * account> </li> <li>
126  *
127  * <a>CreateRepository</a>, which creates an AWS CodeCommit
128  *
129  * repository> </li> <li>
130  *
131  * <a>DeleteRepository</a>, which deletes an AWS CodeCommit
132  *
133  * repository> </li> <li>
134  *
135  * <a>GetRepository</a>, which returns information about a specified
136  *
137  * repository> </li> <li>
138  *
139  * <a>ListRepositories</a>, which lists all AWS CodeCommit repositories associated with your AWS
140  *
141  * account> </li> <li>
142  *
143  * <a>UpdateRepositoryDescription</a>, which sets or updates the description of the
144  *
145  * repository> </li> <li>
146  *
147  * <a>UpdateRepositoryName</a>, which changes the name of the repository. If you change the name of a repository, no other
148  * users of that repository will be able to access it until you send them the new HTTPS or SSH URL to
149  *
150  * use> </li> </ul>
151  *
152  * Branches, by calling the
153  *
154  * following> <ul> <li>
155  *
156  * <a>CreateBranch</a>, which creates a new branch in a specified
157  *
158  * repository> </li> <li>
159  *
160  * <a>DeleteBranch</a>, which deletes the specified branch in a repository unless it is the default
161  *
162  * branch> </li> <li>
163  *
164  * <a>GetBranch</a>, which returns information about a specified
165  *
166  * branch> </li> <li>
167  *
168  * <a>ListBranches</a>, which lists all branches for a specified
169  *
170  * repository> </li> <li>
171  *
172  * <a>UpdateDefaultBranch</a>, which changes the default branch for a
173  *
174  * repository> </li> </ul>
175  *
176  * Files, by calling the
177  *
178  * following> <ul> <li>
179  *
180  * <a>PutFile</a>, which adds or modifies a file in a specified repository and
181  *
182  * branch> </li> </ul>
183  *
184  * Information about committed code in a repository, by calling the
185  *
186  * following> <ul> <li>
187  *
188  * <a>GetBlob</a>, which returns the base-64 encoded content of an individual Git blob object within a
189  *
190  * repository> </li> <li>
191  *
192  * <a>GetCommit</a>, which returns information about a commit, including commit messages and author and committer
193  *
194  * information> </li> <li>
195  *
196  * <a>GetDifferences</a>, which returns information about the differences in a valid commit specifier (such as a branch,
197  * tag, HEAD, commit ID or other fully qualified
198  *
199  * reference)> </li> </ul>
200  *
201  * Pull requests, by calling the
202  *
203  * following> <ul> <li>
204  *
205  * <a>CreatePullRequest</a>, which creates a pull request in a specified
206  *
207  * repository> </li> <li>
208  *
209  * <a>DescribePullRequestEvents</a>, which returns information about one or more pull request
210  *
211  * events> </li> <li>
212  *
213  * <a>GetCommentsForPullRequest</a>, which returns information about comments on a specified pull
214  *
215  * request> </li> <li>
216  *
217  * <a>GetMergeConflicts</a>, which returns information about merge conflicts between the source and destination branch in a
218  * pull
219  *
220  * request> </li> <li>
221  *
222  * <a>GetPullRequest</a>, which returns information about a specified pull
223  *
224  * request> </li> <li>
225  *
226  * <a>ListPullRequests</a>, which lists all pull requests for a
227  *
228  * repository> </li> <li>
229  *
230  * <a>MergePullRequestByFastForward</a>, which merges the source destination branch of a pull request into the specified
231  * destination branch for that pull request using the fast-forward merge
232  *
233  * option> </li> <li>
234  *
235  * <a>PostCommentForPullRequest</a>, which posts a comment to a pull request at the specified line, file, or
236  *
237  * request> </li> <li>
238  *
239  * <a>UpdatePullRequestDescription</a>, which updates the description of a pull
240  *
241  * request> </li> <li>
242  *
243  * <a>UpdatePullRequestStatus</a>, which updates the status of a pull
244  *
245  * request> </li> <li>
246  *
247  * <a>UpdatePullRequestTitle</a>, which updates the title of a pull
248  *
249  * request> </li> </ul>
250  *
251  * Information about comments in a repository, by calling the
252  *
253  * following> <ul> <li>
254  *
255  * <a>DeleteCommentContent</a>, which deletes the content of a comment on a commit in a
256  *
257  * repository> </li> <li>
258  *
259  * <a>GetComment</a>, which returns information about a comment on a
260  *
261  * commit> </li> <li>
262  *
263  * <a>GetCommentsForComparedCommit</a>, which returns information about comments on the comparison between two commit
264  * specifiers in a
265  *
266  * repository> </li> <li>
267  *
268  * <a>PostCommentForComparedCommit</a>, which creates a comment on the comparison between two commit specifiers in a
269  *
270  * repository> </li> <li>
271  *
272  * <a>PostCommentReply</a>, which creates a reply to a
273  *
274  * comment> </li> <li>
275  *
276  * <a>UpdateComment</a>, which updates the content of a comment on a commit in a
277  *
278  * repository> </li> </ul>
279  *
280  * Triggers, by calling the
281  *
282  * following> <ul> <li>
283  *
284  * <a>GetRepositoryTriggers</a>, which returns information about triggers configured for a
285  *
286  * repository> </li> <li>
287  *
288  * <a>PutRepositoryTriggers</a>, which replaces all triggers for a repository and can be used to create or delete
289  *
290  * triggers> </li> <li>
291  *
292  * <a>TestRepositoryTriggers</a>, which tests the functionality of a repository trigger by sending data to the trigger
293  *
294  * target> </li> </ul>
295  *
296  * For information about how to use AWS CodeCommit, see the <a
297  * href="http://docs.aws.amazon.com/codecommit/latest/userguide/welcome.html">AWS CodeCommit User
298  */
