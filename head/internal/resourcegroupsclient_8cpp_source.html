56  * @class ResourceGroupsClient
57  *
58  * @brief Client for AWS Resource Groups
59  *
60  * <fullname>AWS Resource Groups</fullname>
61  *
62  * AWS Resource Groups lets you organize AWS resources such as Amazon EC2 instances, Amazon Relational Database Service
63  * databases, and Amazon S3 buckets into groups using criteria that you define as tags. A resource group is a collection of
64  * resources that match the resource types specified in a query, and share one or more tags or portions of tags. You can
65  * create a group of resources based on their roles in your cloud infrastructure, lifecycle stages, regions, application
66  * layers, or virtually any criteria. Resource groups enable you to automate management tasks, such as those in AWS Systems
67  * Manager Automation documents, on tag-related resources in AWS Systems Manager. Groups of tagged resources also let you
68  * quickly view a custom console in AWS Systems Manager that shows AWS Config compliance and other monitoring data about
69  * member
70  *
71  * resources>
72  *
73  * To create a resource group, build a resource query, and specify tags that identify the criteria that members of the
74  * group have in common. Tags are key-value
75  *
76  * pairs>
77  *
78  * For more information about Resource Groups, see the <a
79  * href="https://docs.aws.amazon.com/ARG/latest/userguide/welcome.html">AWS Resource Groups User
80  *
81  * Guide</a>>
82  *
83  * AWS Resource Groups uses a REST-compliant API that you can use to perform the following types of
84  *
85  * operations> <ul> <li>
86  *
87  * Create, Read, Update, and Delete (CRUD) operations on resource groups and resource query
88  *
89  * entitie> </li> <li>
90  *
91  * Applying, editing, and removing tags from resource
92  *
93  * group> </li> <li>
94  *
95  * Resolving resource group member ARNs so they can be returned as search
96  *
97  * result> </li> <li>
98  *
99  * Getting data about resources that are members of a
100  *
101  * grou> </li> <li>
102  *
103  * Searching AWS resources based on a resource
104  */
