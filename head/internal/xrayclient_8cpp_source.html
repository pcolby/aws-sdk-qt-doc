207  * Uploads segment documents to AWS X-Ray. The X-Ray SDK generates segment documents and sends them to the X-Ray daemon,
208  * which uploads them in batches. A segment document can be a completed segment, an in-progress segment, or an array of

221  *
222  * request> </li> <li>

229  *
230  * request> </li> <li>

240  * <code>in_progress</code>> </li> <li>

243  * segment has been started, but is not complete. Send an in progress segment when your application receives a request that

258  * <code>1</code>> </li> <li>
