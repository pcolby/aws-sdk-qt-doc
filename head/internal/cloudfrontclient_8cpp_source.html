606 /**
607  * Updates the configuration for a web distribution. Perform the following
608  *
609  * steps>
610  *
611  * For information about updating a distribution using the CloudFront console, see <a
612  * href="http://docs.aws.amazon.com/AmazonCloudFront/latest/DeveloperGuide/distribution-web-creating-console.html">Creating
613  * or Updating a Web Distribution Using the CloudFront Console </a> in the <i>Amazon CloudFront Developer
614  *
615  * Guide</i>>
616  *
617  * <b>To update a web distribution using the CloudFront API</b>
618  *
619  * </p <ol> <li>
620  *
621  * Submit a <a>GetDistributionConfig</a> request to get the current configuration and an <code>Etag</code> header for the
622  *
623  * distribution> <note>
624  *
625  * If you update the distribution again, you need to get a new <code>Etag</code>
626  *
627  * header> </note> </li> <li>
628  *
629  * Update the XML document that was returned in the response to your <code>GetDistributionConfig</code> request to include
630  * the desired changes. You can't change the value of <code>CallerReference</code>. If you try to change this value,
631  * CloudFront returns an <code>IllegalUpdate</code>
632  *
633  * error> <b>
634  *
635  * The new configuration replaces the existing configuration; the values that you specify in an
636  * <code>UpdateDistribution</code> request are not merged into the existing configuration. When you add, delete, or replace
637  * values in an element that allows multiple values (for example, <code>CNAME</code>), you must specify all of the values
638  * that you want to appear in the updated distribution. In addition, you must update the corresponding
639  * <code>Quantity</code>
640  *
641  * element> </b> </li> <li>
642  *
643  * Submit an <code>UpdateDistribution</code> request to update the configuration for your
644  *
645  * distribution> <ul> <li>
646  *
647  * In the request body, include the XML document that you updated in Step 2. The request body must include an XML document
648  * with a <code>DistributionConfig</code>
649  *
