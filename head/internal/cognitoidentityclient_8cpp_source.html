67 /**
68  * @class CognitoIdentityClient
69  *
70  * @brief Client for Amazon Cognito Identity
71  *
72  * <fullname>Amazon Cognito</fullname>
73  *
74  * Amazon Cognito is a web service that delivers scoped temporary credentials to mobile devices and other untrusted
75  * environments. Amazon Cognito uniquely identifies a device and supplies the user with a consistent identity over the
76  * lifetime of an
77  *
78  * application>
79  *
80  * Using Amazon Cognito, you can enable authentication with one or more third-party identity providers (Facebook, Google,
81  * or Login with Amazon), and you can also choose to support unauthenticated access from your app. Cognito delivers a
82  * unique identifier for each user and acts as an OpenID token provider trusted by AWS Security Token Service (STS) to
83  * access temporary, limited-privilege AWS
84  *
85  * credentials>
86  *
87  * To provide end-user credentials, first make an unsigned call to <a>GetId</a>. If the end user is authenticated with one
88  * of the supported identity providers, set the <code>Logins</code> map with the identity provider token.
89  * <code>GetId</code> returns a unique identifier for the
90  *
91  * user>
92  *
93  * Next, make an unsigned call to <a>GetCredentialsForIdentity</a>. This call expects the same <code>Logins</code> map as
94  * the <code>GetId</code> call, as well as the <code>IdentityID</code> originally returned by <code>GetId</code>. Assuming
95  * your identity pool has been configured via the <a>SetIdentityPoolRoles</a> operation,
96  * <code>GetCredentialsForIdentity</code> will return AWS credentials for your use. If your pool has not been configured
97  * with <code>SetIdentityPoolRoles</code>, or if you want to follow legacy flow, make an unsigned call to
98  * <a>GetOpenIdToken</a>, which returns the OpenID token necessary to call STS and retrieve AWS credentials. This call
99  * expects the same <code>Logins</code> map as the <code>GetId</code> call, as well as the <code>IdentityID</code>
100  * originally returned by <code>GetId</code>. The token returned by <code>GetOpenIdToken</code> can be passed to the STS
101  * operation <a
102  * href="http://docs.aws.amazon.com/STS/latest/APIReference/API_AssumeRoleWithWebIdentity.html">AssumeRoleWithWebIdentity</a>
103  * to retrieve AWS
104  *
105  * credentials>
106  *
107  * If you want to use Amazon Cognito in an Android, iOS, or Unity application, you will probably want to make API calls via
108  * the AWS Mobile SDK. To learn more, see the <a href="http://docs.aws.amazon.com/mobile/index.html">AWS Mobile SDK
109  * Developer
110  */
