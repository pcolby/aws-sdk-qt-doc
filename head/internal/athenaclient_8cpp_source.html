56  * @brief Client for Amazon Athena
57  *
58  * Amazon Athena is an interactive query service that lets you use standard SQL to analyze data directly in Amazon S3. You
59  * can point Athena at your data in Amazon S3 and run ad-hoc queries and get results in seconds. Athena is serverless, so
60  * there is no infrastructure to set up or manage. You pay only for the queries you run. Athena scales
61  * automatically—executing queries in parallel—so results are fast, even with large datasets and complex queries. For more
62  * information, see <a href="http://docs.aws.amazon.com/athena/latest/ug/what-is.html">What is Amazon Athena</a> in the
