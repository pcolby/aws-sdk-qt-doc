group>

  1. (Optional) Create a Lambda function and a rule that allows CloudWatch Events to invoke your Lambda function when Auto Scaling launches or terminates

    instances>

  2. (Optional) Create a notification target and an IAM role. The target can be either an Amazon SQS queue or an Amazon SNS topic. The role allows Auto Scaling to publish lifecycle notifications to the

    target>

  3. Create the lifecycle hook. Specify whether the hook is used when the instances launch or

    terminate>

  4. If you need more time, record the lifecycle action heartbeat to keep the instance in a pending

    state>
