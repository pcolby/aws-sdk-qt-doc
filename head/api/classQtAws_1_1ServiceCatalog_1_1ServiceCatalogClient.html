Detailed Description

Client for AWS Service Catalog.

<fullname>AWS Service Catalog</fullname>

AWS Service Catalog enables organizations to create and manage catalogs of IT services that are approved for use on AWS. To get the most out of this documentation, you should be familiar with the terminology discussed in AWS Service Catalog
