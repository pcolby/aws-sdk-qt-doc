QString QtAws::SqsOld::SqsResponse::errorString ( ) const
virtual

Get this response's error string.

This override checks for SQS service errors, and if found, returns the first one. Otherwise, this override delegates to the base implementation (which checks for network and XML parse errors, for example).

Also consider using the serviceErrors function to retrieve all of the servive errors (if any), not just the first one (though often SQS will only return one error per response).

Returns
An error string, or a null QString if this response has no errors.

void QtAws::SqsOld::SqsResponse::parseFailure ( QIODevice &  response)
protectedvirtualslot

Parse a failure response.

This implementation parses SQS ErrorResponse elements, as defined by http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl

For example:

<ErrorResponse>
<Error>
<Type>Sender</Type>
<Code>AccessDenied</Code>
<Message>Access to the resource http://sqs.us-east-1.amazonaws.com/ is denied.</Message>
<Detail/>
</Error>
<Error>
...
</Error>
<RequestId>9a285199-c8d6-47c2-bdb2-314cb47d599d</RequestId>
</ErrorResponse>
Parameters
responseResponse to parse.
Note
Unlike most other SQS responses, the SQS ErrorResponse does not wrap the RequestId element in a ResponseMetadata element.
See also
http://queue.amazonaws.com/doc/2012-11-05/QueueService.wsdl
